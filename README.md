# React Snake Game

## Description

A simple react app for a snake game with vite and tailwindcss.

## Getting Started

### Installing dependencies

* How to install
  ```npm install```


### Executing program

* How to run the program
    ```npm run dev``` for development
    ```npm run build``` for production
